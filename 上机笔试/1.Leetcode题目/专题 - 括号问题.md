# 专题 - 括号问题

#### [20. 有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

**示例 1：**

```
输入：s = "()"
输出：true
```

```python
class Solution:
    def isValid(self, s: str) -> bool:
        right = ")}]"
        left = "({["
        dic = {"(":")","{":"}","[":"]"}
        stack = []
        for i in range(len(s)):
            if len(stack) == 0:   ##栈为空
                if s[i] in right:
                    return False
                else:
                    stack.append(s[i])
            else:         ##栈非空
                top = stack[-1]
                if top in right:    ##右括号
                    stack.append(s[i])
                else:             ##左括号
                    if s[i] == dic[top]:
                        stack.pop()
                    else:
                        stack.append(s[i])
        if len(stack) > 0:
            return False
        return True
```

【易错点】 注意最后要对stack做判空操作。
